Git Command Purpose
git tag 1.0 (commit id) Creates a new tag which is a labeled version of a commit
git reset (filename) Unstages the unushed commits that contain the file
git stash list Will list the changes that have not been committed and are stashed with git stash
git stash pop Removes one stashed session from the stash, removing the latest one by default
git rm -f (filename) Stages a deletion of a file, -f overides the up to date check
