Git Command Purpose
git tag 1.0 < commit id > This command creates a tag with the name "1.0" for a specific commit.
git reset < filename > This command reverts local changes on a specific file to a previous commit.
git stash list This command shows saved local changes before the modified files were reverted with git stash.
git stash pop This command takes the local changes used with git stash and reimplements them into the modified files. It also deletes the stash info from git stash list.
git rm -f < filename > This command removes and deletes a specified file from the current repository.
