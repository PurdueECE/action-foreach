git tag 1.0 < commit id> creates, lists,deletes, or verifies a tag object that refers to a commit
git reset < filename> undos local changes of specified file to the state of a git repo
git stash list lists the stash entries of your changes you currently have
git stash pop removes a single stashed state from list and applies to top of current working state
git rm -f < filename> removes files from working tree and index. Also overrides the up-to-date check
