git tag 1.0 <commit id> Assign the commit with the commit id to version 1.0.
git reset <filename> Remove the file from the stage area, which will not be ready to commit.
git stash list Show the latest commit that modifies a file for every file in this branch.
git stash pop Recover the change saved in stash and throw it away. It will merge with the current commit if exists.
git rm -f <filename> Remove a file from the staging area and working directory by force.
