git tag 1.0 <commit id> This command is used to add a tag named 1.0 in git that doesn't change and is used as a marking.
git reset <filename> This command is used to undo changes of a specified file.
git stash list This git command is used to view all stashes you have created. Stashes in git are temporary changes that you've made on a file that aren't committed.
git stash pop This command is used to reapply past changes that were stashed.
git rm -f <filename> This command is used to remove a specified file. The -f portion of the command is to override a Git safety check.
