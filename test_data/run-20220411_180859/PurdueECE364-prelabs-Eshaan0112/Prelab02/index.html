Command Function
git tag 1.0 *commit id* Tags the commit id with the version 1.0. Unlike branches tags do not change
git reset *filename* Unadds filename from staging area
git stash list Lists the stash entries that you currently have.
git stash pop Removes a single stashed state from the stash list and apply it on top of the current working tree state
git rm -f *filename* Force removes a file from both the repository and the working directory
