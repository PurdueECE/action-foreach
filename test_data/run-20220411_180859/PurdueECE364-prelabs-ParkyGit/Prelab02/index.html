Command Purpose
git tag 1.0 <commit id> Tag an older commit using the ID of the commit in git. This allows us to easily find it later if we want to roll back or keep track of milestones.
git reset <filename> Git reset allows you to unstage files from a commit. In this case, this would unstage the file corresponding to the filename.
git stash list Git stash saves all of your current un-pushed progress, and resets your directory back to a clean state. Git stash list simply lists all of your previous git stashes.
git stash pop Git stash pop will apply the stashed insertions/deletions to your current workspace.
git rm -f <filename> This command will remove <filename> from the directory and untracks it on the git repo.
