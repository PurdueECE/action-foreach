git tag 1.0 <commit id> Creates a tag for <commit id> called 1.0. Tags are references that point to specific points in Git history
git reset <filename> Moves the repository back to the commit specified in <filename>
git stash list Lists the current stash entries. A stash is a recorded state of the working directory and index
git stash pop Removes a single stashed state from the stash list and applies it on top of the current working tree state
git rm -f <filename> Removes <filename> from the index. The -f flag overrides the up-to-date check
