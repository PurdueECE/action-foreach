IMPORTANT GIT COMMANDS TABLE

Command Explaination
git tag 1.0 <commit id> This can be used to assign version 1.0 to the commit specified by the commit id
git reset <filename> Remove the specified file from the staging area, but leave the working directory unchanged. This unstages a file without overwriting any changes.
git stash list git stash can be used to save uncommited changes and get back the original file. git stash list can be used to see a list of all the stashes in a list form
git stash pop This command can be used to remove the most recent stash from the stack of stashes and apply them.
