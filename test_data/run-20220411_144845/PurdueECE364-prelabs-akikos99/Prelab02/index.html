Command Purpose
git tag 1.0 < commit id > Tags specific points of a repository for importance.
git reset < filename > Returns the file back to the previous commited state.
git stash list Lists all of the current stashes you have.
git stash pop Reapplys your stashed work back to your working area.
git rm -f < filename > Removes file from path as well as the Working Directory and the Staging Index.
