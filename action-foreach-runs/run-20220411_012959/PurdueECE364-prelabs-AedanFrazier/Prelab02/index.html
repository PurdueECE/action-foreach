git tag 1.0 <commit id> Puts a "tag" on a specific commit. Makes a past commit easier to lookup in the future.
git reset <filename> A way to roll back your files to a previous commit.
git stash list A way to list the different stashes of a working directory.
git stash pop A way to delete a stash and roll back your files.
git rm -f <filename> Removes a file from your repository.
